### Introduction
Welcome to ***AnimeHub***, a website where you can download Anime in HD for free. The aim of this repository is to allow visitors to download their favourite Anime for free. There are several websites (9Anime, AniWatch, etc) that allow streaming Anime but they don't allow downloading.

---

### Sources
We use an M3U8 tool to download all of the Anime sources. For the sources found on YouTube, we use a simple MP4 downloader. This allows us to download each episode in HD quality and store the video files on Google Drive. Here is a full list of websites where we get the sources from:  

- 9Anime
- AniWatch
- BiliBili
- Crunchyroll
- YouTube
